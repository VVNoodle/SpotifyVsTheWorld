FAQ

What does this app do?

You know how Twitch and YouTube Livestreams have realtime viewer counts? This is like that but for Spotify. This app shows the number of users also listening to the same artist you are currently listening to.

How is this different than Spotify Charts?

This app updates leaderboards in realtime and counts number of listeners at a given time. This also tracks artists instead of tracks.

Is this Mac only?

Yes. At least for now.

How do you count tracks with multiple artists?

We count the Artist which "hosts" the track you're playing.

How do you count remixes?

The artist being remixed gets the count.
